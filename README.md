# WILDCORE Prototype

This repository now contains a playable Python prototype for **WILDCORE**:

- persistent terrain transformation across waves
- elemental combo interactions
- enemy adaptation forecasting with fairness lead-time
- post-forecast mitigation options to preserve player agency
- hybridization taxes and volatility acceleration
- biome seeding + commander identity per run
- wave-by-wave run simulation with story moments and end-of-run summaries

## Run

```bash
python wildcore_game.py
```

## Test

```bash
python -m pytest -q
```
